Highway to Wipe

Application de gestion de guilde World of Warcraft

Plateforme complète pour optimiser la coordination de guilde
• Gestion de roster avec rôles et compositions de raid
• Calendrier d'événements et système d'inscription
• Farming collaboratif avec tracking des ressources
• Intégration API Warcraft Logs pour analyses de performances
• Intégration API Raider.io pour suivi des progressions Mythique+
• Notifications temps réel avec Action Cable
• Dashboard personnalisé par rôle (tank, heal, DPS)

Ruby on Rails 7 PostgreSQL Stimulus Turbo API REST Action Cable
